# Uptime Kuma

This folder should contain screenshots of the Uptime Kuma monitoring dashboard.

## Required Screenshots

### 1. uptime-kuma.png

Capture showing:

- Main dashboard with all monitors
- Status indicators (up/down)
- Response times
- Uptime percentages

## How to Capture

1. Navigate to `https://uptime.yourdomain.com`
2. Take a screenshot of the main dashboard

## Screenshots to Capture

### 1. Dashboard Overview

Shows all monitors and their current status:

- Green checkmarks for online services
- Red X for offline services
- Response time metrics
- Uptime percentage

### 2. Monitor Details

Individual monitor information:

- Heartbeat history
- Response time graphs
- SSL certificate status

### 3. Status Page

Public status page view:

- Custom branded status page
- Service availability
- Incident history

### 4. Alert Settings

Notification configuration:

- Discord/Telegram webhook setup
- Email notifications
- Alert rules

## Image Requirements

- **Format**: PNG or JPG
- **Resolution**: Minimum 1280x720
- **Style**: Include both light and dark themes if applicable

## Naming Convention

- `uptime-kuma-dashboard.png` - Main dashboard
- `uptime-kuma-monitor.png` - Single monitor view
- `uptime-kuma-status.png` - Public status page
- `uptime-kuma-alerts.png` - Alert configuration
