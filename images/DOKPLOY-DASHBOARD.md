# Dokploy Dashboard

This folder should contain screenshots of the Dokploy deployment platform.

## Required Screenshots

### 1. dokploy-dashboard.png

Capture showing:

- Main dashboard overview
- List of deployed applications
- Resource usage overview
- Recent deployments

## How to Capture

### Method 1: Screenshot Tool

1. Navigate to `https://dokploy.yourdomain.com`
2. Use your system's screenshot tool:
   - **Linux**: `Print Screen` key or `gnome-screenshot`
   - **macOS**: `Cmd + Shift + 4`
   - **Windows**: `Win + Shift + S`

### Method 2: CLI Screenshot

```bash
# Using scrot (Linux)
sudo apt install scrot
scrot -u dokploy-dashboard.png
```

## Recommended Views to Capture

1. **Dashboard Overview** - Main page with application list
2. **Application Detail** - Single app deployment view
3. **Deployment Logs** - Build and deploy output
4. **Environment Variables** - Configuration page
5. **Domain Settings** - SSL and routing configuration

## Image Requirements

- **Format**: PNG or JPG
- **Resolution**: Minimum 1280x720
- **Quality**: High (no compression artifacts)

## Naming Convention

- `dokploy-dashboard.png` - Main dashboard
- `dokploy-app-detail.png` - Application details
- `dokploy-logs.png` - Deployment logs
- `dokploy-settings.png` - Settings page
